Stack tecnologico: Database: MySQL | Backend: Express.js | Frontend: React

Sviluppo di un sito e-commerce responsive per la vendita di prodotti tech. Il progetto include:
- Una homepage con una lista dei prodotti piu venduti,
- Uno shop con lista completa di prodotti, impaginazione e filtri di ricerca per promozioni, brand, S.O. e prezzo,
- Una pagina di dettaglio del singolo prodotto con correlati,
- Una wish list,
- Un carrello con gestione delle quantita e calcolo del prezzo totale,
- Un pop-up per l'iscrizione alla newsletter.
